# 自行实现的的曼哈顿多边形布尔运算
路线：  
输入多边形->初始化拓扑关系->基于网格法求交点->交点插入轮廓(以及处理特殊交点)->嵌套关系处理->绕边算法得到结果

## 1.前提条件
1. 输入为曼哈顿多边形，多边形无内孔、无自交
2. 多边形的边为线段，多边形的点精度为int
3. 输出多边形也无内孔

## 2.支持场景
1. 支持交、并、差布尔运算
2. 输出支持曼哈顿多边形集（平面上无相交关系的多个独立曼哈顿多边形）， 显然，若输入是求两个不相交多边形的并，输出必然也是两个多边形，所以用集来表示输出
3. 输入也可以为曼哈顿多边形集（得确保这个集内两两不相交）

## 3.当前不支持但可能的扩展
1. 场景:输入无内孔，输出带内孔
2. 场景:输入也带内孔  
这两种场景都需进一步扩展代码实现

```
    ———— MPolygon.h 曼哈顿多边形轮廓结构体
    ———— MPolygonSet.h 曼哈顿多边形集结构体
solve/ 布尔运算核心处理
    ———— MBSOCore.h 
    ———— MBSOCore.cpp 布尔运算核心处理逻辑
```
